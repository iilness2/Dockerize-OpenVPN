# Dockerize-OpenVPN
This repo to Dockerized OPENVPN which can deploy as a Docker especially at AWS ECS 

# environment variable you can modified
This is environment variable you can modified to adjust with your needs
```
PRIV_IP
PRIV_SUBNET_MASK

PUBLIC_IP

PROTOCOL_OPENVPN

VPN_PORT

MYSQL_ROOT_PASS

MYSQL_USER

MYSQL_PASS

KEY_SIZE

CA_EXPIRE
```
